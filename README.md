# xgjplus

班级小管家工具

## 描述

`xgjplus` 是一个用于管理班级信息的工具，支持从多个接口获取学生信息、成绩等，并生成 Excel 文件。

## 安装

```bash
npm install -g xgjplus
```

## 使用

### 构建项目

在使用前需要先构建项目：

```bash
npm run build
```

### 运行项目

构建完成后，可以通过以下命令运行项目：

```bash
xgjplus
```
